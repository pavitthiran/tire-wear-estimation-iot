# Tire Wear Estimation with Privacy Using Advanced Learning Methods

This project is an IoT-based final year engineering project designed to estimate tire wear in real time using multiple sensors and secure data transmission techniques. The system focuses on vehicle safety, predictive maintenance, and user data privacy.

## Academic Details
- Degree: B.Tech Computer Science Engineering
- Institution: Vel Tech Rangarajan Dr. Sagunthala R&D Institute of Science and Technology
- Year: 2025
- Project Type: Final Year Major Project

## Technologies Used
- Arduino / ESP8266
- BMP180 Pressure & Temperature Sensor
- MPU6050 Accelerometer
- LCD Display (I2C)
- Wi-Fi Communication
- AES Encryption
- SHA1 Password Hashing
- HTML (Web Dashboard)

## System Features
- Real-time sensor data acquisition
- Tire pressure and motion monitoring
- Encrypted data transmission
- Login-protected admin dashboard
- LCD-based local output display
- Privacy-focused system architecture

## Project Modules
1. Sensor Module (Pressure, Gyroscope, Motion)
2. Microcontroller Module (Arduino + ESP8266)
3. Security & Encryption Module
4. Web Dashboard & Admin Login
5. Cloud Communication Layer

## Security Implementation
- AES encryption for sensor data
- SHA1 hashing for login authentication
- Brute-force protection for admin login

## Folder Structure
- arduino-code : Embedded system source code
- documentation : Project report
- images : Hardware setup and output screenshots

This project was developed for academic purposes to demonstrate IoT, security, and privacy-preserving system concepts.
